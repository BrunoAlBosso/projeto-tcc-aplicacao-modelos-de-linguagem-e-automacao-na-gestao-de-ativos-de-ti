# APLICAÇÃO DE MODELOS DE LINGUAGEM E AUTOMAÇÃO INTELIGENTE NA GESTÃO DE ATIVOS DE TI

> Trabalho de Conclusão de Curso (TCC)

## Sobre o Projeto

Este repositório contém o código-fonte do sistema (Nexus CMDB).

### Principais Funcionalidades

* **Dashboard Interativo:** Visualização em tempo real de ativos e incidentes.
* **Automação com IA:** Geração de relatórios analíticos de infraestrutura utilizando Agentes de IA (integrado via n8n).
* **Agente Conversacional (RAG):** Chatbot para consulta e gestão de ativos via linguagem natural.
* **Gestão de Incidentes:** Registro e vinculação de falhas aos itens de configuração.

---

## Tecnologias Utilizadas

* **Frontend:** React.js e Vite.
* **Backend:** Node.js (Servidor Local) e Supabase (BaaS - PostgreSQL).
* **Automação & IA:** n8n (Workflow Automation) e Google Gemini API.
* **Infraestrutura de Teste:** Ansible (Coleta de fatos).

---

## Pré-requisitos

Para executar este projeto localmente, é necessário ter instalado em sua máquina:

* **[Node.js]**.
* **[Visual Studio Code]**.
* Gerenciador de pacotes **npm**.
